## tabulergm

<!-- badges: start -->
[![R-CMD-check](https://github.com/gvegayon/tabulergm/actions/workflows/R-CMD-check.yaml/badge.svg)](https://github.com/gvegayon/tabulergm/actions/workflows/R-CMD-check.yaml)
<!-- badges: end -->

`tabulergm` is an R package designed to translate models estimated with the
[`ergm`](https://cran.r-project.org/package=ergm) framework into
publication-ready tables and explanatory summaries.

> **Note**: This package is under active development. The current version
> contains the baseline package structure. Functionality will be added in
> subsequent releases.

## Installation

```r
# Install from GitHub
# install.packages("remotes")
remotes::install_github("gvegayon/tabulergm")
```
